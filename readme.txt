== Description ==
The Easy Related Random Posts plugin is a versatile tool that allows you to display related or random posts on your WordPress website with just a few simple steps. Whether you want to keep your visitors engaged by showing related content or add an element of surprise with random posts, this plugin has you covered.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/easy-related-random-posts` directory or install it through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to the "Easy Related Random Posts" settings in your WordPress dashboard to configure the plugin.

== Frequently Asked Questions ==
= How do I display related or random posts on my website? =
You can use the `[easy_related_random_posts]` shortcode to display related or random posts in your posts or pages. Simply add the shortcode to the content editor where you want the posts to appear.

Example Shortcode:
[easy_related_random_posts]


= Can I customize the appearance and behavior of the related or random posts section? =
Yes, you can customize the plugin settings in the "Easy Related Random Posts" settings page. You can set the headline text, post limit, display type (related or random posts), and even control the caching behavior.

= What is the cache time option, and how does it work? =
The cache time option allows you to control how often the related or random posts are refreshed. You can set a cache time in seconds. A value of 0 disables caching, while any positive value specifies the duration of caching.
